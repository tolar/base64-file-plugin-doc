category: news

We’ve released **v4.0.0 of the Base64 File Plugin**, bringing full compatibility with the **JetBrains Platform 2025.3**.

Along with this release, we’ve added a new [Versioning](/versioning) page to the documentation. This page explains how plugin versions map to JetBrains Platform versions.
